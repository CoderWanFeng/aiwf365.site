# AI 短剧《回声》全流程

> 一支 11 分钟的 AI 短剧，从剧本到成片全流程拆解。所有步骤都在 M2 Mac 上跑通，平均单集成本 ≈ ¥3。

## 一、前期准备

在动手之前，请确认你的本地环境：

- **ComfyUI** ≥ 0.3.x，本地部署
- **Kling / 可灵** API Key（生成视频）
- **ChatTTS** 或 **GPT-SoVITS**（配音）
- **剪映 / CapCut** 客户端（最终剪辑）

## 二、Step 1：剧本（30 分钟）

先用 GPT-4o 写剧本骨架，prompt 模板：

```text
你是资深短剧编剧，请写一个 {题材} 的 {时长} 分钟短剧剧本。
要求：
1. 主角必须有一个明显的人性弱点
2. 反转要在第 8 分钟出现
3. 结尾留 1 句金句
输出格式：场景序号 / 场景描述 / 对白 / 镜头建议
```

拿到剧本后再人工改一轮台词，去掉 AI 味。

## 三、Step 2：分镜 + 关键帧（1 小时）

每个场景用 **Midjourney** 出 4 张关键帧，统一画风：

```
--ar 16:9 --style raw --s 200 --v 6.1
```

把所有图拖进 ComfyUI，用 **IPAdapter + ControlNet** 锁定角色长相。

## 四、Step 3：图生视频（2 小时）

调用 Kling API，参数：

- 模式：标准
- 时长：5s / 段
- 运镜：自动跟随

每段视频原始 1080p、24fps，存到 `clips/scene_NN.mp4`。

## 五、Step 4：配音 + 字幕

用 **GPT-SoVITS** 训练一个固定主播音色（约 30 分钟样本即可），批量合成所有对白。

字幕用剪映「智能字幕」一键导出 SRT。

## 六、Step 5：剪辑 + 调色

剪映时间轴：

```
00:00 ─ 00:30   钩子（前 3 秒冲突）
00:30 ─ 08:00   主体
08:00 ─ 10:30   反转
10:30 ─ 11:00   金句
```

调色统一用 **LUT：复古港风** 即可。

## 七、踩坑记录

- ❗ Kling 5s 段不要拼成 8s，会有卡顿
- ❗ 字幕字体用思源黑体 Bold，不要用系统默认
- ❗ 配乐别用 YouTube Audio Library，版权风险大
